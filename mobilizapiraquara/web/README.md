# Mobiliza Piraquara — Web

App Next.js 14 (App Router) do portal.

## Scripts

```bash
npm run dev      # http://localhost:3000
npm run build    # prisma generate + next build
npm run db:seed  # dados de exemplo
```

## Vercel

- **Root Directory do projeto:** `mobilizapiraquara/web`
- `vercel.json` define `framework: "nextjs"` para evitar deploy como site estático
